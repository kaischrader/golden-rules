Common Rules
Global rules which should be obeyed for the application.
  1. Consistency: If you are using an element more than once, it is expected that you use the element in a consistent way.
    1. The look (UI) of the element should be consistent.
    2. The handling (UX) of the element should be consistent.
  2. Call to action: Give users a clear indication what to do next.
  3. Offer the user more than one way to find and execute (important) actions.
    1. For unexperienced or new users offer descriptive texts, symbols, tooltips and other helpful means.
    2. Experienced or power users prefer a fast way to execute actions. Here keyboard shortcuts can be appropriate.
  4. Offer users a context sensitive help
  5. Visual hierarchy
